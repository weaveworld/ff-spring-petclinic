Finding Vets

page.open(URL);
Vets can be found by clicking the `Veterinarians` button 

page.image().dash('Veterinarians');
The `Veterinarians` page contains the list of the professionals 
page.get('Veterinarians').click()

page.image().dash('css=h2');

The page header is `Veterinarians` 

page.get('css=h2').toString() ?? 'Veterinarians'

In the example the first name on the list is `James Carter`

page.get('vets').get('css=td').toString() ?? 'James Carter'

Paging

Pages can be shown by clicking the page number or a paging icon 

page.image().dash('css=div.container-fluid>div>div');

At first, the first page can not be clicked but the second 

page.get('1').css('text-decoration') ^?? ['underline']
page.get('2').css('text-decoration') ?? ['underline']

Forward

One can get the next page by clicking the next page icon 
page.get('css=.fa-step-forward').image().scale(0.7);

page.image().dash('css=.fa-step-forward');
page.get('css=.fa-step-forward').click()
After a forward, the first page can be clicked but the second can not be clicked 
page.get('1').css('text-decoration') ?? ['underline']
page.get('2').css('text-decoration') ^?? ['underline']

Backward

Clicking the backward icon the next page appears 
page.get('css=.fa-step-backward').image().scale(0.7);

page.image().dash('css=.fa-step-backward');
page.get('css=.fa-step-backward').click()
Now the first page can be clicked but the second 
page.get('1').css('text-decoration') ^?? ['underline']
page.get('2').css('text-decoration') ?? ['underline']
